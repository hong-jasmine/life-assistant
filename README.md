# ⚡ 極速卡片.Flash cards | LIFE-ASSISTANT | 未來生活指揮中心

這是一個專為追求效率與科技美學設計的個人生活管理工具。整合財務管理、任務追蹤與數據分析。

## 🌌 系統特徵 (Features)

- **📈 智能數據監測**：即時追蹤收入與支出，具備高對比度的視覺反饋。
- **📝 任務遙測系統**：管理每日待辦事項，確保所有目標都在控制範圍內。
- **🌑 科技感 UI/UX**：基於深色模式優化的毛玻璃質感、霓虹發光邊框與流暢動畫。
- **📊 數據匯出**：支持 JSON/CSV 格式，輕鬆備份你的生活數據。

## 🛠 技術架構 (Tech Stack)

- **Core:** HTML5 / CSS3 / JavaScript (ES6+)
- **Styling:** CSS Variables (Modern Dark Palette), Glassmorphism Effects
- **Icons:** Lucide Icons / FontAwesome

## 🚀 快速啟動 (Quick Start)

1. **複製本倉庫**
   ```bash
   git clone https://github.com/hong-jasmine/life-assistant.git
   ```

2. **進入目錄**
   ```bash
   cd life-assistant
   ```

3. **開啟專案**
   
   直接在瀏覽器中打開 `index.html` 即可體驗。
